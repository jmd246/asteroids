# Asteroids

A small Python arcade game built with Pygame.

## Overview

This project is a simple asteroid shooter where the player controls a ship, avoids incoming asteroids, and survives as long as possible.

## How to Play

- Use the arrow keys or WASD to move the ship.
- Press the space bar to fire.
- Avoid collisions with asteroids and try to survive as long as you can.

## Project Goals

This game is a fun way to practice game development concepts such as:

- sprite-based movement and collisions
- simple enemy spawning
- game loop timing
- basic scoring and survival mechanics

